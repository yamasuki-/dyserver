# 画面一覧

## 1. ヘッダー (共通)
- アプリケーションタイトル (リンク: エンドポイント一覧へ)
- ナビゲーションメニュー
    - エンドポイント一覧
    - ログ一覧
    - 全体設定 (ログ保存ON/OFF等)

## 2. エンドポイント一覧画面 (トップページ)
### 概要
作成済みのエンドポイント設定を一覧表示し、管理する画面。

### 機能・表示要素
- **新規作成ボタン**: エンドポイント作成画面へ遷移。
- **設定インポートボタン**: JSONファイルから設定を読み込む。
- **設定エクスポートボタン**: 現在の設定をJSONファイルとしてダウンロード。
- **設定リセットボタン**: 全設定を初期状態に戻す。
- **エンドポイントリスト**:
    - パス (例: `/api/v1/users`)
    - 有効なメソッド (GET/POST/PUT/DELETE バッジ表示)
    - 操作アクション:
        - 編集: エンドポイント編集画面へ遷移。
        - 削除: エンドポイント設定を削除。

## 3. エンドポイント作成・編集画面
### 概要
特定のエンドポイントの挙動（メソッド、レスポンス、切替ロジック）を設定する画面。

### 機能・表示要素
- **基本設定**:
    - パス入力フィールド (例: `api/users/[id]`)
    - 説明 (任意)
- **メソッド設定** (GET/POST/PUT/DELETE ごとにタブ切り替え等で設定):
    - **有効化スイッチ**: メソッド自体の有効/無効。
    - **レスポンスセット管理**:
        - レスポンスセット追加・削除・複製。
        - 各セットの設定:
            - セット名 (管理用)
            - ステータスコード (200, 400, 500等)
            - レスポンスヘッダー (Key-Value追加)
            - レスポンスボディ (JSON/Textエディタ)
            - 応答なし設定 (チェックボックス)
    - **切替ロジック設定**:
        - モード選択 (Default / Random / Conditional)
        - 条件設定 (Conditional選択時):
            - 対象 (Header / Body / Query)
            - キー名
            - マッチする値と対応するレスポンスセットの紐付け
- **保存ボタン**: 設定を保存して一覧へ戻る。
- **キャンセルボタン**: 変更を破棄して一覧へ戻る。

## 4. ログ一覧画面
### 概要
受信したリクエストと、それに対するレスポンスの履歴を確認する画面。

### 機能・表示要素
- **フィルタリング・検索エリア**:
    - エンドポイントパス (セレクトボックス or テキスト)
    - メソッド (セレクトボックス)
    - ステータスコード
    - キーワード検索 (Body/Header/Query内の検索)
    - 特定キーフィルタ (Key, Value入力)
- **ログリスト (テーブル表示)**:
    - 日時
    - メソッド
    - パス
    - ステータスコード
    - リクエスト概要 (Bodyの一部等)
    - 操作: 詳細表示ボタン
- **一括操作**:
    - ログ削除 (全削除 / フィルタ結果削除 / 選択削除)
    - エクスポート (CSV / JSON / Excel)
- **ページネーション**: ページ送り機能。

## 5. ログ詳細画面 (モーダルまたは別ページ)
### 概要
個別のログの詳細情報を表示する。

### 機能・表示要素
- **基本情報**: 日時, パス, メソッド, ステータス, 所要時間。
- **Request情報**:
    - Headers (全表示)
    - Query Parameters
    - Body (JSON整形表示 / Raw表示)
- **Response情報**:
    - Headers
    - Body (JSON整形表示 / Raw表示)
    - 適用されたレスポンスセット情報

## 6. 全体設定画面
### 概要
アプリケーション全体の振る舞いを設定する画面。

### 機能・表示要素
- **ログ設定**:
    - リクエストログ保存: ON/OFF
    - レスポンスログ保存: ON/OFF
- **データ管理**:
    - 全データリセット (初期化)
